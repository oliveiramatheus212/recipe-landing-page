# Recipe Landing Page

This repository contains a simple static landing page showcasing a cupcake recipe. It was developed as a practical exercise during the **Fullstack Developer** course by **Rocketseat**.

## 📁 Project Structure

```
recipe-landing-page/
│
├── index.html        # Main HTML file with recipe content
├── style.css         # Styles using CSS variables and Google Fonts
└── assets/           # Image files used in the page
    ├── main-image.png
    ├── bg-image.png
    └── ph_heart-fill.png
```

## 🌟 Features

- Responsive layout centered in the viewport
- Uses custom Google font (`Alice`) for a warm, handwritten feel
- CSS variables for easy color scheme adjustments
- Recipe sections for ingredients and preparation steps

## 🚀 How to Use

1. Clone or download this repository.
2. Open `index.html` in a web browser (double-click or use `Live Server` in VS Code).
3. Enjoy the cupcake recipe and feel free to personalize the styles or content!

> This is a static project — no build steps, dependencies, or server required.

Created as part of the Rocketseat Fullstack Developer track, this landing page was a hands-on challenge to practice HTML structure, CSS styling, and organization of assets.
